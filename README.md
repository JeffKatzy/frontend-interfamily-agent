# IFS Frontend

This is the frontend for [Interfamily Agent](https://github.com/JeffKatzy/interfamily-agent). It is built with Next.js and Tailwind CSS.

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
